Responda as perguntas abaixo:

Na tag input com atributo type="text", para que serve o atributo "placeholder"?

<>R:<>é utilizado para indicar ao navegador o texto que deverá aparecer escrito no campo do formulário

Na tag input com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

<>R:<>para indicar se o item está selecionado por padrão

Na tag input para que serve o atributo "value"?

R:Você deve usar o atributo value para definir o valor a ser enviado por este item

Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

R:Devemos usa o select,junto com atributo option

Para que serve a tag label? Para que serve o atributo "for" nessa tag?

R:A tag label é um elemento HTML que representa uma legenda para melhorar a acessibilidade de um item de interface do usuário. O atributo for faz um tipo de link entre a label e o input. Logo o for tem que apontar para o ID do input

Na tag option para que serve o atributo "selected"?

R:Para seleciona a opção que o usuário escolhera

Nas tags de formulário para que serve o atributo "required"?

R:usado para indicar que um determinando campo de formulário é obrigatório para o envio do mesmo. Ao adicionar esse atributo a um campo de formulário, o navegador obriga o usuário a inserir dados naquele campo antes de enviar o formulário

Na tag form para que serve o atributo "action"?

R:O atributo action define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados

Na tag form para que serve o atributo "method"?

R:este método faz com que o conteúdo do formulário seja codificado exatamente como no método GET, mas ao invés de anexá-lo à URL, ele é enviado ao servidor em um bloco de dados endereçado à URL

Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?

R:Esse valor pode ser substituído por um formmethodatributo em um button ou input elemento.

Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?

R:post: Corresponde ao método HTTP POST ; os dados do formulário são incluídos no corpo do formulário e enviados ao servidor.
